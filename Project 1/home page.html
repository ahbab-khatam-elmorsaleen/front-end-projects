Write Once, Run Anywhere

Java is a general-purpose modern programming language initially developed by Sun Microsystems, and currently owned by Oracle Corporation.


The key feature of the language is platform independence: it means that the same Java program can be run on multiple platforms without any changes!


This principle is also known as "write once, run anywhere" (or WORA).
